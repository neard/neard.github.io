Installation

  • Download and install Neard.
  • If you already have installed Neard, stop it.
  • Download a MySQL version of your choice.
  • Extract archive in neard\bins\mysql\. Directory structure example :
[-] neard
  | [-] bins
  |  | [-] mysql
  |  |  | [-] mysql5.5.24
  |  |     | neard.conf
  • Start Neard.
  • Switch to the MySQL version you have extracted on Neard :

Infos

Starting in version 8.0, MySQL is only available on 64-bit platforms.
