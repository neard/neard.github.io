Installation

  • Download and install Neard.
  • If you already have installed Neard, stop it.
  • Download a PHP version of your choice.
  • Extract archive in neard\bins\php\. Directory structure example :
[-] neard
  | [-] bins
  |  | [-] php
  |  |  | [-] php5.2.17
  |  |     | neard.conf
  • Start Neard.
  • Switch to the PHP version you have extracted on Neard :

If you have the warning icon on your PHP version, you will have to switch the Apache version first.
