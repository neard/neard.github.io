Installation

  • Download and install Neard.
  • If you already have installed Neard, stop it.
  • Download a Git version of your choice.
  • Extract archive in neard\tools\git\. Directory structure example :
[-] neard
  | [-] tools
  |  | [-] git
  |  |  | [-] git1.8.4
  |  |     | neard.conf
  • Edit the neard.conf file and replace the key gitVersion with the correct version.
  • Start Neard.
