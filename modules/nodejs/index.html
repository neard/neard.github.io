About

Node.js : an open-source, cross-platform runtime environment for developing server-side web applications.

  • https://nodejs.org/

Installation

  • Download and install Neard.
  • If you already have installed Neard, stop it.
  • Download a Node.js version of your choice.
  • Extract archive in neard\bins\nodejs\. Directory structure example :
[-] neard
  | [-] bins
  |  | [-] nodejs
  |  |  | [-] nodejs0.10.22
  |  |     | neard.conf
  • Start Neard.
  • Switch to the Node.js version you have extracted on Neard :
